Shisha no teikoku

•Titre original : 屍者の帝国 The Empire of Corpses

•Origine : Roman

•Date de sortie : 02/10/2015 · Saison : Automne 2015

•Genres : Drame - Historique - Horreur / Épouvante - Mystère - Psychologique - Science-fiction - Surnaturel

•Thèmes : Littérature - Mort - Scientifique - Steampunk

•Studio d'animation : Wit Studio

Lien streaming

Description :

L'histoire de Shisha no Teikoku se déroule dans l'Europe du 19e siècle, et s'articule autour de John Watson avant qu'il ne soit le partenaire de Sherlock Holmes. Il est repéré par le gouvernement pour devenir un agent secret. Cependant, Watson vit dans un monde où les Frankenstein ne sont pas une légende. Ce sont des cadavres humains qui sont re-conçu avec une fausse âme afin de les utiliser comme main d'oeuvre pour améliorer le développement industriel.

Notre avis :

Pas franchement facile à suivre mais un thème que je trouve original les personnages sont plutôt bien développés. Quant à la musique l'orgue n'est pas franchement mon fort, mais sinon elle est belle. J'aime tout particulièrement l'ending !! Côté visuel, c'est super beau ! Les couleurs sont superbes, le dessin est épuré. Quant à l'animation, elle est très bien menée !
